# Network Traffic Analysis & Threat Detection using Wireshark

## Overview
This project demonstrates real-time network traffic capture and analysis using Wireshark.
The goal is to simulate SOC analyst activities and identify normal and malicious network behavior.

## Tools Used
- Wireshark
- Windows / Kali Linux
- Nmap (optional)

## Key Analysis
- ICMP traffic analysis
- DNS request/response analysis
- HTTP traffic inspection
- Credential sniffing detection
- Port scan identification

## Skills Gained
- Packet analysis
- Network security monitoring
- SOC investigation techniques
- Wireshark filtering

## Conclusion
This project helped build practical SOC-level skills by analyzing live network traffic
and identifying potential security threats.
